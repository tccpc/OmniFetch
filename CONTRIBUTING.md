# 贡献指南

感谢你对 OmniFetch 感兴趣！我们非常欢迎社区的贡献。

## 如何贡献

1. **Fork 本仓库**：点击右上角的 Fork 按钮，将项目复制到你自己的 GitHub 账户。
2. **克隆项目**：将 Fork 后的项目克隆到本地。
   ```bash
   git clone https://github.com/your-username/OmniFetch.git
   ```
3. **创建分支**：为你的修改创建一个新的分支。
   ```bash
   git checkout -b feature/your-feature-name
   ```
4. **提交修改**：完成代码修改后，提交你的更改。
   ```bash
   git commit -m "feat: 添加了某个新功能"
   ```
5. **推送到远程**：将分支推送到你的 GitHub 仓库。
   ```bash
   git push origin feature/your-feature-name
   ```
6. **提交 Pull Request**：在 GitHub 上发起 Pull Request，描述你的修改内容。

## 开发规范

- 请保持代码风格一致。
- 提交信息请使用清晰的描述（建议使用 Conventional Commits 规范）。
- 如果添加了新功能，请确保更新 README.md 中的相关说明。

## 反馈问题

如果你发现了 Bug 或有新的功能建议，欢迎在 Issues 中提出。
